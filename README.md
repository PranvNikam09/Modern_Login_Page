# 🚀 Modern Login & Registration Page

A sleek, state-of-the-art authentication interface featuring smooth dual-form transitions and a premium aesthetic. This project is built with high attention to detail, focusing on smooth animations and a responsive user experience.

---

## ✨ Features

- **Interactive Dual-Form:** Seamlessly switch between "Sign In" and "Sign Up" views with a fluid sliding animation.
- **Micro-Animations:** Engaging hover effects and button transitions for a premium feel.
- **Modern Typography:** Optimized for readability using the **Montserrat** font.
- **Vibrant Design:** Uses high-contrast linear gradients and a clean card-based layout.
- **Social Login ready:** Includes integrated icons for popular social platforms (Google+, Facebook, GitHub, LinkedIn).
- **Fully Responsive:** Adapts beautifully to different screen sizes.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Advanced animations, Flexbox layout, and custom gradients.
- **JavaScript (Vanilla):** Light logic for form toggling and interaction.
- **Font Awesome:** High-quality icons.
- **Google Fonts:** Premium typography.

---

## 🚀 Getting Started

Follow these simple steps to run the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PranvNikam09/Modern_Login_Page.git
   ```
2. **Navigate to the project folder:**
   ```bash
   cd Modern_Login_Page
   ```
3. **Launch the project:**
   Simply open `index.html` in your favorite web browser.
